# 🖼️ AI Image Detector

An AI-powered web application that detects whether an uploaded image is AI-generated or real.

🌐 Live Demo: https://ai-image-detect.lovable.app

---

## 🚀 Features

- 📤 Upload any image
- 🤖 Detect whether the image is AI-generated or real
- ⚡ Fast prediction
- 📱 Responsive UI
- 🎨 Clean and modern interface

## 🛠️ Tech Stack

- Frontend: React
- TypeScript
- Tailwind CSS
- Lovable
- AI Detection API / Model

## 🎯 How it Works

1. Upload an image.
2. The image is sent to the AI detection model/API.
3. The model analyzes image patterns.
4. The prediction is displayed as:
   - ✅ Real Image
   - 🤖 AI Generated Image
